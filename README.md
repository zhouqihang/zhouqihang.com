# README.md

## install

1. `git clone git@github.com:zhouqihang/zhouqihang.com.git`

2. `composer install`

3. `npm install`

## config

1. rename `.env.example` to `.env`, then change your config.

2. run `php artisan key:generate` set the application key.

3. `php artisan migrate`

## useage

1. npm run start
